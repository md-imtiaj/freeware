Uninstallation Instructions for Software:

 

How to uninstall software in Windows 10

  • Click the Cortana search icon
  • Type or say "Programs and Features"
  • From the list of software displayed, right-click the software title and select "Uninstall"

How to uninstall software in Windows 8 / 8.1

  • From the Metro, Right click anywhere around the applications and a bar will pop up from the bottom.
  • Click on the "All Apps" Icon in the right bottom corner of the bar
  • In the applications menu, Right Click on the program you want to uninstall
  • Click on the bar that pops up from the bottom with an Uninstall Icon
  • Click on "Uninstall" again to confirm the uninstallation.

How to uninstall software in Windows Vista / Windows 7

  • From the Start Menu, select Control Panel.
  • Under Programs click the Uninstall a Program link.
  • Select the program you wish to remove and right click then select "Uninstall/Change".
  • Follow the prompts to finish the uninstallation of the software.

How to uninstall software in Windows XP

  • From the Start Menu, select Control Panel.
  • Click the Add or Remove Programs icon.
  • Select the program you wish to remove and click on the "Remove/Uninstall" button.
  • Follow the prompts to finish the uninstallation of the software.

Please Note:

How to remove add-ons / extensions from your web browser.

Chrome

  • Click the "customize and control" button in Google Chrome and select Settings.
  • Select Extensions in the left column menu.
  • A list of all extensions will be shown. Click the trash can icon to remove the extension.

Firefox

  • Click on the Tools menu and select Add-ons.
  • Go to the add-on you wish to disable and click the Disable or Uninstall button.

Internet Explorer

  • Click on the Tools menu and select Manage add-ons.
  • Select the add-on you wish to disable and then click Disable at the bottom of the page.
